# Azumi + Dracon-Platform — Master TODO

## North Star

A full-stack Rust framework for people tired of JS frontend fragility. One language, one compiler, one type system — from database to DOM. No WASM. No hydration. HTML is truth.

## Non-Goals

- ORM integration (let users pick sqlx/diesel/sea-orm)
- Auth package (too many variants; signed state + demo is enough)
- CSS framework integration (compile-time validation is the right level)
- Switch from Axum (best Rust web framework, 0.8 is stable)
- File-system routing scanner (macro is simpler, type-safe)
- Client-side routing (MPA with Idiomorph is simpler and faster)
